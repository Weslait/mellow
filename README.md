# mellow
HETIC - Projet - Semaine Intensive

Suivez ces étapes pour prévisualiser les fichiers HTML du projet sur votre machine locale.

### Prérequis

* Node.js installé sur votre ordinateur.

### Étapes pour démarrer le serveur

1. Ouvrez votre terminal ou invite de commande.
2. Accédez au répertoire racine du projet.
3. Lancez le serveur avec la commande suivante :

node index

4. Une fois le serveur démarré, ouvrez votre navigateur web.
5. Accédez à l'URL suivante pour voir la page d'accueil : http://localhost:3000/landing/index.html
